# MNIST Digit Classifier

This project is a handwritten digit recognition system using Python, TensorFlow/Keras, and Streamlit.

## Features
- CNN-based MNIST digit recognition
- Interactive web interface using Streamlit
- API endpoint for predictions

## How to Run Locally
1. Clone the repo: `git clone https://github.com/Tecmayank/mnist-digit-classifier.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app/app.py`
4. Open the URL shown in your browser

## Technologies
- Python
- TensorFlow / Keras
- Streamlit
